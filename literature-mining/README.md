## 📖 这是什么 / What Is This

`literature-mining` 是一个专注于文献阅读方法论的技能工具包。它不负责检索文献（已有 `literature-review`、`literature-reviewer-skill` 等优秀技能覆盖检索），而是专注于检索之后的**阅读、分析、比较和提炼**环节。

`literature-mining` is a methodology toolkit focused on literature reading. It does NOT search for papers (that's covered by `literature-review`, `literature-reviewer-skill`, etc.) — instead, it focuses on **reading, analyzing, comparing, and synthesizing** papers you've already found.
